# Thomson's Auto Body Repair Website

A professional, fully responsive multi-page website for Thomson's Auto Body Repair services.

## Features

- **Responsive Design**: Optimized for desktop, tablet, and mobile devices
- **Fixed Header**: Professional navigation header with logo on all pages
- **Mobile Menu**: Hamburger dropdown menu for mobile devices
- **Search Bar**: Integrated search functionality
- **Page Transitions**: Smooth rolling circle spinner animation between pages
- **Professional Branding**: Dark blue and gold color scheme matching the company logo
- **Multiple Pages**: Home, About, Services, Projects, Shop, and Contact pages

## Pages

1. **Home (index.html)** - Welcome page with service highlights
2. **About (about.html)** - Company story, mission, vision, and values
3. **Services (services.html)** - Complete list of all repair services
4. **Projects (projects.html)** - Portfolio of completed repairs
5. **Shop (shop.html)** - Available products and parts
6. **Contact (contact.html)** - Contact information and inquiry form

## Files

- `index.html` - Home page
- `about.html` - About page
- `services.html` - Services page
- `projects.html` - Projects page
- `shop.html` - Shop page
- `contact.html` - Contact page
- `style.css` - Main stylesheet with responsive design
- `script.js` - JavaScript for navigation and animations
- `assets/images/logo.svg` - Company logo

## How to Use

1. Extract the ZIP file to your desired location
2. Open `index.html` in your web browser
3. Navigate between pages using the menu
4. The logo in the header links back to the home page from any page
5. Use the search bar to search for services

## Customization

To customize the website:
- Edit `style.css` to change colors, fonts, or layout
- Edit HTML files to update content
- Replace `assets/images/logo.svg` with your own logo
- Update contact information in all pages as needed
